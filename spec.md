# Hello World Web App

## Overview
A simple web application that displays a "Hello World" greeting with colorful rectangular boxes.

## Features

### Display Elements
- Large "Hello World" text displayed prominently at the top of the page
- Five rectangular boxes displayed below the text
- Each box has a different color: red, blue, green, yellow, and purple

### Layout and Responsiveness
- Boxes are evenly spaced horizontally on larger screens
- On small screens, boxes stack vertically for better mobile viewing
- Clean, centered layout with appropriate spacing

### Technical Requirements
- Frontend-only application (no backend data storage required)
- Responsive design that adapts to different screen sizes
- Static content with no user interaction or data persistence needed

## User Interface
The application presents a simple, clean interface with the greeting text as the focal point and the colored boxes serving as visual elements below it.
